The Game of Life

This page / game came from the enjoyable and enlightening Udemy class - Build the Game of Life. I've since extended the code to make it more user friendly.

The Game of Life is a simulation of "Life". It's a basic simulation of life and death; highlighting the effects of isolation and over-population. You can and should read more about the origions of the game and the rules behind the simulation to understand how this game / page works.
